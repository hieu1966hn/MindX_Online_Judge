# Tổng hai số

## Đề bài

Cho hai số nguyên `a` và `b`. Hãy tính và in ra tổng của chúng.

## Định dạng Input

Một dòng duy nhất chứa hai số nguyên `a` và `b`, cách nhau bởi một dấu cách.

## Định dạng Output

In ra một số nguyên duy nhất là tổng `a + b`.

## Ràng buộc

- `-10^9 ≤ a, b ≤ 10^9`

## Ví dụ

### Input mẫu 1

```
3 5
```

### Output mẫu 1

```
8
```

### Giải thích

`3 + 5 = 8`
